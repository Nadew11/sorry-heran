# Photos

Drop your photos in this folder named **1.jpg, 2.jpg, 3.jpg, 4.jpg, 5.jpg**.

They'll automatically show up in the "Us" gallery. Until then, each slot shows a "📷 add a photo" placeholder.

Tip: any of those 5 photos from your phone work great — just rename them to `1.jpg` … `5.jpg` and put them here, then commit + push.
